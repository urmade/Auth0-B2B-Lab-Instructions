In order to set you up for success, we need a few things in place before we can start:
1
Please register a new tenant on https://auth0.com. You won't need to provide anything but an email and a password, which will give you a free Auth0 tenant with a 3 week enterprise trial enabled (which includes features we will use throughout the lab). We would recommend to set up the tenant in the Europe region.
2
If you are following an instructor-led training, you should have received an AWS Cloud9 environment which already contains the complete lab environment.

Otherwise Make sure you have Node.js Version 16 or higher installed and a code editor ready. We will use Visual Studio Code for the live coding, but any other editor that you feel comfortable with should be good as well. Lastly make sure you have cloned the lab repository to your local machine.
Once you are set up, feel free to start with the first lab around integrating Auth0 into your application. To the next lab
