Welcome to the Auth0 B2B SaaS Lab module! This is a hands-on lab that will walk you through all the steps necessary to secure an application with a multi-tenant capable solution that allows for separation of data as well as for user collaboration. The lab consists of five modules:

  1. Setting up Auth0
  2. Customizing the login form
  3. Enabling business to business authentication
  4. Customizing the login flow
  5. Enabling collaboration
Before you start, there are a few things that you have to set up. Get started right away by heading to the "Prerequisite" module!
